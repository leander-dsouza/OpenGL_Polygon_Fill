# OpenGL_Polygon_Fill
OpenGL code to fill a user-specified polygon

* Install OpenGL:

      sudo apt-get install libglu1-mesa-dev freeglut3-dev mesa-common-dev

* To compile the program:

      g++ main.cpp -o PolygonFill -lglut -lGLU -lGL

* To execute the program:

      ./PolygonFill
